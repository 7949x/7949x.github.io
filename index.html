7949x: Crowdsourcing Assignments

7949x is a website for sharing assignments for programming class. Simply fill out the form below with the required information and your assignment will be uploaded to the site shortly after, available for all to download. Created by students, for students.


All assignments will be uploaded in a ".txt" format which can then be copied and pasted into your code editor of choice, likely either visual studio or notepad. You can use "Command + F" on Mac or "Ctrl + F" on Windows to search within this webpage for assignments listed in the downloads.
